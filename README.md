# HashJing

**HashJing** — это генеративный арт-проект, исследующий границу между случайностью и порядком через визуальное представление 256-битных хэшей. Проект вдохновлён бинарной логикой, криптографией и символикой *Ицзина* — древнекитайской Книги Перемен.

![изображение](https://github.com/user-attachments/assets/142b76fb-0793-4ea7-b17f-7201e08be5f9_)

*Мандала, построенная из SHA256-хэша собственного названия проекта "HashJing":*  
`0x2b054d225d025fc24c58030bda276b16089ae21cc8aff93d2751472a98eab06c`

## Что это такое

HashJing преобразует 256-битные строки (например, хэши Ethereum, Bitcoin или криптографически безопасные seed’ы) в **мандалы** — круговые диаграммы с 64 секторами и 4 радиальными слоями.
Каждый сектор соответствует одной hex-цифре, каждый слой — одному биту.
Таким образом, визуализация полностью и однозначно кодирует весь хэш.

Мандала становится формой для размышления — об энтропии, симметрии, редкости и визуальном представлении вероятности.

## Основные идеи проекта

* Бинарные паттерны как визуальный язык
* Случайность как наблюдаемая структура
* Параллели с гексаграммами и принципами *Ицзина*
* Хэш как печать, отпечаток выбора
* Полная детерминированность: один хэш → одна форма

## Техническая реализация

HashJing совместим с платформами вроде **Art Blocks**.

Хэш может быть получен:

* как `token_hash` при выпуске токена,
* как off-chain seed из криптографически стойкого генератора (`secrets`, VRF),
* или задан пользователем вручную.

> Под «чистой энтропией» в проекте понимается 256-битный seed, полученный из безопасного, непредсказуемого источника.

Визуализация строго детерминирована: одинаковый хэш всегда даёт одинаковую мандалу.
